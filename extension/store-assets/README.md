# Store Assets

This folder contains images for the Chrome Web Store listing.

## Required Images

| File | Size | Purpose |
|------|------|---------|
| `promo-small.png` | 440×280 px | Small promotional tile (required) |
| `screenshot-create.png` | 1280×800 px | Screenshot: Create tab view |
| `screenshot-library.png` | 1280×800 px | Screenshot: Library tab view |

## Upload Instructions

When submitting to the Chrome Web Store Developer Dashboard:
1. **Store Icon**: Use `../icons/icon128.png` (already 128×128)
2. **Small Promo Tile**: Upload `promo-small.png` 
3. **Screenshots**: Upload both screenshot files

## Tips for Better Screenshots

The generated mockups above are placeholder illustrations. For the BEST store listing:
- Take **real screenshots** of the extension running in Chrome
- Use Chrome DevTools to capture at exactly 1280×800 px
- Show the side panel open next to a real Google Flow project
