## 🖥️ DEMO (0:25–2:00)

### General Settings

> "At the top, you have the general settings:"

> **Video Model** — Pick your default model. Right now you've got Veo 3.1 Fast (fastest), Veo 3, and others as Google adds them. The refresh button pulls the latest models from your Flow page.

> **Video Ratio** — Landscape (16:9), Portrait (9:16), or Square (1:1). This sets the default for new queues.

> **Videos per task** — How many generations per prompt. Set to 1 for drafts, 2-3 for variety.

> **Stop on error** — Toggle this on if you want the queue to halt when a prompt fails. Leave it off for unattended runs.

**[SCREEN: Walk through each setting in General section]**

### Timing

> **Wait time** — The pause between prompts (in seconds). Default is 10–20 seconds. AutoFlow picks a random value in this range to look natural.

> **Typing Mode** — When enabled, AutoFlow types your prompt character by character instead of pasting all at once. This mimics human behavior. Turn it on if you want extra safety.

**[SCREEN: Show Timing section → toggle Typing Mode ON]**

### Downloads

> **Auto-download videos** — When ON, videos download automatically as they finish generating. No scanning needed.

> **Video Resolution** — Choose the default download quality: 720p (original, free), 1080p, or 4K.

> **Auto-download images** — Same thing for images, with selectable resolution.

> "Pro tip: disable 'Ask where to save' in Chrome settings for seamless downloads."

**[SCREEN: Show Downloads section → toggle auto-download → resolution picker]**

### Interface

> **Language** — Match this to your Google Flow interface language. AutoFlow uses this to find buttons and labels correctly.

**[SCREEN: Show Language dropdown → English selected]**

### Maintenance

> **Clear Flow Cache** — If generation stalls or the page gets slow, this clears Flow's internal cache. One click fix.

**[SCREEN: Show red "Clear Flow Cache" button]**
